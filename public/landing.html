WebMCP Inspector

Inspect & augment WebMCP tools on any page

WebMCP Inspector lets you discover, inspect, and debug tools exposed via the WebMCP spec on any web page in real time.

Open Inspector

Discover Tools

Automatically scans pages for HTML forms and imperative tools registered via navigator.modelContext.

Inspect Metadata

View and validate WebMCP attributes like toolname, tooldescription, and parameter annotations.

Augment Forms

Add or edit MCP metadata on any form to expose it as a tool, with overrides saved per URL.

Live Preview

See the target page rendered in a sandboxed Webfuse session alongside your edits.
